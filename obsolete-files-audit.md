# Аудит устаревших файлов (Obsolete Files Audit)

В данном документе перечислены кандидаты на удаление из репозитория Very Well Decor Studio, результаты поиска упоминаний в коде и принятые решения.

## Кандидаты на удаление

### 1. `assets/images/hero1.webp`
* **Путь**: `assets/images/hero1.webp`
* **Описание**: Изображение обложки главного экрана (размер ~371 КБ).
* **Причина проверки**: В стилях и разметке в качестве активного фона используются файлы `newhero.webp` (для десктопа) и `newhero-mobile.webp` (для мобильных). Данный файл предположительно остался от старой версии дизайна.
* **Результаты поиска упоминаний**:
  * `Get-ChildItem -Recurse | Select-String "hero1.webp"` нашел единственное упоминание:
    * `partials/head-common.html:22`: `<link rel="preload" href="/assets/images/hero1.webp" as="image" type="image/webp">`
  * Никаких упоминаний в CSS (`style1.css`), JavaScript (`main.js`), HTML-файлах страниц или конфигурациях не обнаружено.
* **Решение**: Удалить из репозитория после исправления тега `preload` в `partials/head-common.html` на актуальный `/assets/images/newhero.webp`.
* **Уровень риска**: Крайне низкий (файл не используется в верстке или логике).

## Файлы, сохраненные намеренно

* **`assets/images/newhero.webp` и `newhero-mobile.webp`**: Активно используются в CSS (`style1.css`) в качестве фонового изображения первого экрана.
* **`assets/images/hero_decor_1.webp`**: Используется в качестве изображения для Open Graph (`og:image`) в шапке.
* **`api/send-notification.js`**: Реализует обработку отправки уведомлений с форм (интегрирована с Vercel/API). Сохранена без изменений.
* **`.nojekyll`**: Необходим для корректного хостинга на GitHub Pages (предотвращает игнорирование папок с символом подчеркивания, например, `_astro` или `_nuxt`, хотя для Vite это менее критично, но полезно для предотвращения проблем с ресурсами).
* **`vercel.json`**: Конфигурационный файл Vercel, содержащий правило `cleanUrls`. Необходим для продакшена.
